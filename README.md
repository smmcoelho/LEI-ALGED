# Collections and Data Structures

This repository contains simple and clear examples demonstrating the usage of **collections** and **basic data structures** in **C#**.

The goal is to provide easy-to-understand code samples that illustrate:
- When to use different collection types
- Basic operations on collections
- Common patterns when working with data in C#

## Further Reading
